CyberArk Station Suspended Tool

CyberArk Station Suspended Tool Summary

The CyberArk Station Suspended Tool is a utility that provides authorized users the ability to check to see if an account is locked within CyberArk. Typically, the user will report the following error: Station Suspended for User: XXXXXXXXXX.

If the user's account is locked within CyberArk, the utility has the ability to unlock the account.

CyberArk Station Suspended Tool has following properties:

  • TEXT AREA:                          Enter the user's ID you want to check in the text area
  • FULL NAME:                        This read-only field will display the user's first and last name
  • NETWORK ID:                      This read-only field will display the user's network ID
  • USER SUSPENDED:              This read-only field will display the user's suspended status with CyberArk
  • FAILED LOGINS:                  This read-only field will display the number of failed logins to CyberArk
  • SEARCH BUTTON:               Once a user's ID has been entered into the text area, the Search button will become enabled. (please note: the results are NOT displayed on the form instantly. It may take a few seconds.)
  • ACTIVATE BUTTON:              If a user's ID is locked within CyberArk, the Activate button will appear and allow you unlock the user's account. (please note: the results are NOT displayed on the form instantly. It may take a few seconds.)
  • EXIT BUTTON:                   The Exit button is used to exit out of the utility. (please note: if you clicked on the Help button and the help page is still open, exiting the utility will NOT close the help window)
  • VAULT COMMUNICATION STATUS:               Displays green if the utility can ping the CyberArk vault and red if the utility cannot ping the CyberArk vault.
  • ALLOWED SOURCE:              Displays green if the utility is being executed from an allowed source and red if the utility is NOT being executed from an allowed source.
  • ACTIVE DIRECTORY CMDLETS:                   Displays green if the utility detects the Active Directory PowerShell CMDlets are present and red if the Active Directory PowerShell CMDlets are NOT present.

Usage

To check if a user's account is locked out within CyberArk, simply enter the user's network ID in the TEXT AREA of the form. Then click the SEARCH button.

The results of the search will be displayed within the utility. (please note: the results are NOT displayed on the form instantly. It may take a few seconds.)

Unlock User's Account

If the user's account is locked within CyberArk, the ACTIVATE button will be displayed.

Click the ACTIVATE button to unlock the user's account and reset the number of failed logins to zero (0) within CyberArk.

Note USER SUSPENDED is now false and failed logins is now zero (0).

Click the EXIT button to exit the CyberArk Station Suspended Tool.
